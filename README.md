# Musical Journeys — Riddhi Sanyal

A complete music teaching website covering 10 instruments with dual Indian Sargam and Western Staff Notation throughout.

## Live Site
Deploy to GitHub Pages → **https://riddhisan.github.io/mj/**

## Pages Included
- **Home** — instrument gallery, dual notation philosophy
- **Piano** — keys, scales, both notations
- **Strings** — Guitar, Ukulele, Mandolin, Banjo (with Raga Bhairavi for strings)
- **Harmonica** — 10-hole diatonic diagram, blow/draw convention + Sargam + Staff
- **Melodica** — keyboard + breath
- **Bansuri** — Indian bamboo flute, pentatonic raga progression
- **Western Flute** — Boehm system, staff notation
- **Violin** — Western style tab + Indian classical style (tabs)
- **Indian Classical Ragas** — classified by instrument suitability
- **Beginner Chapter** — SARGAMs, 2/3/4-note progressions, Alankars
- **Indian Songs** — Bollywood, Folk, Bhajan, Patriotic, Rabindra Sangeet
- **Western Songs** — Classical, Waltz, Country, Folk, Jazz, Soft Rock
- **Books** — 8 published books with Amazon links
- **About** — Full author biography
- **Contact** — Email form → Riddhi.Sanyal@gmail.com

## GitHub Pages Setup
```bash
git init

git commit -m "Musical Journeys website v1"
git remote add origin https://github.com/riddhisan/mj.git
git push -u origin main
```
Then: GitHub → repo Settings → Pages → Source: main / root
